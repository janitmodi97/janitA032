TAG DEFINITIONS

Section 1 : Meta tag
Metadata is data (information) about data. The meta tag provides metadata about the HTML document. Metadata will not be displayed on the page, but will be machine parsable. Meta elements are typically used to specify page description, keywords, author of the document, last modified, and other metadata.
Section 2 : Head tag
The element is a container for all the head elements. The element can include a title for the document, scripts, styles, meta information, and more.
Regular DIV element
ASIDE - Some information that relates to the main topic, i.e., related posts.
Section 3 : Title tag
The title tag is an HTML title element critical to both SEO and user experience that is used to briefly and accurately describes the topic and theme of an online document. The title tag is displayed in two key places: Internet Browser – Title Tags display in the top bar of internet browsers.
Section 4 : Body tag
The body tag defines the document's body. The element contains all the contents of an HTML document, such as text, hyperlinks, images, tables, lists, etc.
Section 5 : Section tag
The section tag defines sections in a document, such as chapters, headers, footers, or any other sections of the document.
Section 6 :Div tag
The div tag is used to group block-elements to format them with CSS.
Section 7 : Aside tag
The HTML aside tag is an HTML 5 element that defines a section that is tangentially related to the content around it in the HTML document. This tag is also commonly referred to as the aside element.
Section 8 : Header tag
Heading tags are part of the HTML coding for a Web page. Headings are defined with H1 to H6 tags. The H1 tag defines the most important heading on the page (usually the largest or boldest, too), whereas H6 indicates the lowest-level heading.
Section 9 : Footer tag
The footer tag defines a footer for a document or section. A footer element should contain information about its containing element. A footer element typically contains: authorship information.
Section 10 : Nav tag
The nav tag defines a set of navigation links. Notice that NOT all links of a document should be inside a nav element. The nav element is intended only for major block of navigation links.
